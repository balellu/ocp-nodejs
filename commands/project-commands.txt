Kubectl commands
---- ---- ---- -
Creates:-
~~~~~~~~
kubectl --kubeconfig kube-files/kubeconfig-eks -n ude-demo-app apply -f kube-files/ude-demoapp-deploy.yaml 

Deletes:-
~~~~~~~~
kubectl --kubeconfig kube-files/akubec delete -n ude-demo-app

Gets:-
~~~~~~
kubectl --kubeconfig kube-files/akubec get nodes
kubectl --kubeconfig kube-files/akubec get svc -n ude-demo-app
